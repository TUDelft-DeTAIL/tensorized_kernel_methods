# Docker

Build: `docker build -t jtsch/tkm . `

Build and Push: `docker build -t jtsch/tkm . && docker push jtsch/tkm`

Pull: `docker pull jtsch/tkm`

Run with terminal: `docker run -it jtsch/tkm`
